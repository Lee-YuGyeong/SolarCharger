# SolarCharger
- Demo Video : https://youtu.be/aiDhyKgqYNw <br />

## Introduction
“SolarCharger”은 태양광과 일반 전력을 효율적으로 제어하며 전력을 사용하는 하이브리드형 충전 시스템이다. <br />

1. 고속 충전과 동시 충전을 최대 3대까지 지원한다. <br />
2. 태양광 배터리가 완전히 충전이 되면 태양광을 사용하는 방식을 사용하여 태양광과 일반 전력을 효율적으로 제어한다. <br />
3. 효율적인 충전소 사용을 위한 애플리케이션을 지원한다. <br />
4. 충전소가 설치된 현재 날씨를 크롤링하여 보여준다. <br />
5. 충전 케이블 현황을 실시간으로 보여준다. <br />
6. 현재 사용 중인 전력원을 확인할 수 있다. <br />
7. 고장 난 포트를 신고 하고 확인할 수 있다. <br />
8. 관리자 모드로 진입하면 태양광 예측 발전량과 1시간 주기의 태양광 발전량을 확인할 수 있다. <br />

## Development Environment
- 태양광 패널 : 54cmX28cm 피크전력 40W 최대전압 18V
- 태양광컨트롤러 : 30A
- 라즈베리파이 : Raspberry Pi 3 B+
- mysql
- Apache
- php
- Android Studio (java+kotlin)

## Application Version
- minSdkVersion : 16
- targetSdkVersion : 30
